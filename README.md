# Booking and Utility Automation System

## Description

This is a simple Java console-based application that provides:

* Room booking functionality.
* Electricity bill calculation utility.
* Menu-driven interface for easy interaction.

The project is developed using Core Java and can be run on any JDK without additional libraries or frameworks.

## Features

* Book a room by entering customer details and number of days.
* Automatically calculate the booking amount.
* Calculate electricity bills based on units consumed.
* User-friendly menu-driven system.

## Technologies Used

* Java
* JDK 8 or above
* Scanner class for user input
